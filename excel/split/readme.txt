1. 将输入文件放到src文件夹里
2. 在excel_split.py文件中设置文件路径、分表和sheet对应的两个列标（从0开始）
3. 打开命令行终端（在windows任务栏的搜索框搜索cmd），进入到代码文件（excel_split.py）所在目录下，运行: python excel_split.py
4. 查看src文件夹，如果无误，已生成对应结果文件
5. 用到的库是openpyxl，如果安装的python里没有，需要在运行代码前先进行安装，打开cmd输入并回车：pip install openpyxl
6. 想要学习更多excel的操作方法，可查阅技术文档https://openpyxl.readthedocs.io/en/stable/tutorial.html
